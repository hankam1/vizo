======================================================
  УСТАНОВКА И ЗАПУСК
======================================================

1. УСТАНОВКА PYTHON-ЗАВИСИМОСТЕЙ
   Открой терминал в папке automation и выполни:

   pip install -r requirements.txt

2. УСТАНОВКА БРАУЗЕРА ДЛЯ PLAYWRIGHT
   После предыдущего шага выполни:

   playwright install chromium

3. ПЕРВЫЙ ЗАПУСК
   python main.py

   При первом запуске откроется браузер — войди в свой аккаунт
   Claude (claude.ai), затем вернись в терминал и нажми Enter.
   Авторизация сохранится в файл claude_auth.json и больше
   не потребуется.

4. ПОСЛЕДУЮЩИЕ ЗАПУСКИ
   python main.py

   Браузер откроется уже авторизованным автоматически.

======================================================
  ЧАСТЫЕ ПРОБЛЕМЫ
======================================================

• "Не найдено поле ввода Claude"
  → Сайт claude.ai обновил интерфейс. Сообщи — исправим селекторы.

• "Транскрипт не найден"
  → Видео без субтитров. Попробуй другое видео или добавь субтитры.

• Озвучка зависла
  → Проверь баланс в боте озвучки. Длинные тексты (~35 000 симв.)
    могут обрабатываться 3–5 минут — это нормально.

• Claude закрылся до ответа
  → Увеличь timeout в modules/claude_ui.py (параметр timeout=600).
